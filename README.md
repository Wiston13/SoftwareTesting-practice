本 repository 為個人練習 QA 測試流程所建立的簡單測試練習。

- 測試案例撰寫
- 測試場景設計
- 缺陷回報
- 測試工具操作

使用工具：

- Jira
- Zephyr
- Trello
- Google Sheets
